# Need for Speed Underground 2

## About

This repository contains a copy of **Need for Speed Underground 2** preserved for players who want to enjoy this classic racing experience.

The goal of this project is to make the game easier to access for the community and help preserve one of the most iconic titles in the Need for Speed franchise.

Enjoy the streets of Bayview, customize your cars, and relive the underground racing era.

---

## Game Information

**Title:** Need for Speed Underground 2  
**Release Year:** 2004  
**Genre:** Racing / Street Racing  
**Developer:** EA Black Box  
**Publisher:** Electronic Arts

- Car customization
